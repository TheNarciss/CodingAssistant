# CodingAssistant

A lightweight autonomous coding agent with a web UI and a sandboxed workspace. It plans tasks, chooses tools, executes actions, and streams logs and results to the frontend in real time.

## Features
- Planner/reviewer/optimizer pipeline with safe tool execution
- Web search and page fetching tools
- Sandboxed file system for generated code
- WebSocket server for streaming logs and answers
- React + Vite frontend

## Project Structure
- app/graph: workflow nodes and routing
- app/tools: file, web, and terminal tools
- app/state: state definitions
- server.py: FastAPI WebSocket server
- main.py: CLI runner
- PlaygroundForCodingAssistant: sandbox workspace
- frontend/: UI client

## Requirements
- Python 3.9+
- Node.js 18+

## Setup
1. Create a virtual environment and install Python dependencies.
2. Install frontend dependencies.
3. Start the backend and frontend.

## Run
Backend:
- python server.py

Frontend:
- npm install
- npm run dev

## Notes
All files generated by the agent are written under PlaygroundForCodingAssistant.
